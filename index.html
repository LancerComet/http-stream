HttpStream

HttpStream is now a full-featured Stream implementation that supports the Web Streams API!

Original Demo

In this example, what we are basically doing is:

  • Create two `HttpStream`s from both a png image and a jpg image.
  • Read their pixel size without downloading them completely.

New Stream API Features

HttpStream now supports:

  • ReadableStream interface - Standard Web Streams API compatibility
  • Chunked reading - Read files in configurable chunks with backpressure support
  • Stream methods - getReader(), read(), pipeTo(), pipeThrough()
  • Error handling - Proper HTTP error handling and stream error propagation
  • Backward compatibility - Original readAsync() and getLength() methods still work

Please open the console to see what is happening, and remember to check the size of downloaded content in "Network" panel.
